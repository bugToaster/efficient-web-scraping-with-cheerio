

# Efficient Web Scraping with Cheerio


### Setup 

To setup this project run

```bash
  npm install
  cp .envsample .env
  ```
### Run
Modify the value in .env and run following command
  ```bash
  node app.js
```

### Features

- Customizable Rules
- Data Store in XLSX
- Data Validation
- Error Handling
- Custom Scripts
- Identify Google adds
- Pagination Support



### Tech Stack

**Backend:** Node, Cheerio, xlsx
